# ucla-stats-152-final-project-topic2

## Can you predict next-season temperature from large-scale climate patterns?

## Dataset: ERA5 monthly 2m temperature (course dataset) plus 2-3 publicly available climate indices such as Niño 3.4 (course dataset), the Pacific Decadal Oscillation (PDO), and the Arctic Oscillation (AO), all downloadable from NOAA PSL.

Build a regression model that predicts seasonal-mean temperature anomalies one season
ahead at a chosen location (or region) from current-season climate indices. How well does the
model generalize? Use multiple regression (assess confounding among the indices), perform
appropriate out-of-sample validation of the predictions, and assess residuals.
