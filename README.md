# Topic 2 Temperature Prediction MVP

This is a very early MVP for the Stats 152 final project.

The main idea is to explore whether large-scale climate indices may help explain or predict future temperature patterns. This version is not the final analysis. It only loads the data, makes simple EDA plots, and lists a basic plan for the project.

## Data

- ERA5 monthly 2m temperature
- Nino 3.4 monthly index
- PDO monthly index
- AO monthly index

## Simple Plan

1. Load the temperature data and climate index data.
2. Make simple plots and summary tables.
3. Choose a simple temperature outcome after talking with the professor.
4. Later, try a basic regression model.
5. Later, add validation and residual checks.
